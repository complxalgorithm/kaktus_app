Want to Contribute?

You Can Add Code

Since Kaktus is open source, you can contribute any code you feel is necessary. I would definitely prefer, however, if you consult with me first before you change/add anything. Feel free to contribute code using any language, framework, or library you want, too! Not only would it benefit this application, but it would benefit my brain, as well.


All you need is the GitHub link and some coding skills. Happy contributing!


Donate

This application is completely free to use. It's only logical to keep everything free. I'll put in my own time to make it; you can quickly convert text into a document type of your choice and upload it practically anywhere you have access. As long as you're happy, I am, too.


Money is indubitably always nice, though. If you find that this application is worth some money, feel free to donate to my PayPal below! That'd be neat ;)
